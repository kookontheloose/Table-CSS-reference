attribute selector [attribute] {} can be used to target HTML elements that already contain attributes. Elements of the same type can be targeted differently by their attribute or attribute value. This alleviates the need to add new code, like the class or id attributes. [href] { color: magneta; }
adding type and/or attribute values [src*'winter']

or [href*'winter']		 Elements of the same type can be targeted differently by their attribute or attribute value img src='/images/seasons/cold/winter.jpg'

css img[src*='winter'] { height: 50px; }

looks for an img element with an attribute of src that contains the string 'winter', and sets the height to 50px.
